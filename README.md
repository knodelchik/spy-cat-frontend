# Spy Cat Agency - Dashboard

A modern, minimalist dashboard for managing Spy Cat Agency agents. Built with **Next.js 14 (App Router)** and **TailwindCSS**.

## Features

- **Agent Management:** View, recruit (add), and decommission (delete) spy cats.
- **Live Salary Editing:** Inline editing of agent salaries directly from the card.
- **Smart Validation:**
  - Prevents numeric input in names.
  - Validates required fields.
  - Interactive feedback via Toast notifications.
- **Clean UI:** Minimalist "Black & Zinc" aesthetic with responsive design.

## Tech Stack

- **Next.js** (React Framework)
- **TypeScript**
- **TailwindCSS** (Styling)

## Getting Started

1. Clone the repository:

    git clone <YOUR_FRONTEND_REPO_URL>
    cd spy-cat-frontend

2. Install dependencies:

    npm install

3. Run the development server:

    npm run dev

4. Open your browser:
   Navigate to http://localhost:3000

> Note: Ensure the Backend API is running on port 8000 for data fetching to work correctly.
